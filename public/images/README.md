# Images Folder

Drop your images here and reference them with `/images/filename.ext`

## Expected files:
- `profile.jpg` — your profile/avatar photo (used in the Hero section)
- `project-1.jpg`, `project-2.jpg`, etc. — project screenshots (used in Projects section)

## Usage in Hero
Once you add `profile.jpg`, uncomment the `<Image>` block in `components/Hero.js`
and remove the placeholder div above it.
